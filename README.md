# Mountain Lake Biological Station - R Users Group

| Week # | Topic | Aims |
| --- | --- | --- |
| 1 | - [How to install R and R Studio](https://www.r-bloggers.com/2022/01/how-to-install-and-update-r-and-rstudio/) <br> - [Getting started in R and R Studio](https://rafalab.dfci.harvard.edu/dsbook/getting-started.html) | - Differentiate R vs. R Studio <br> - Learn how to set up R and R Studio on your personal computer <br> - Familiarize yourself with some of the main panels/windows in R Studio |
| 2 | - [R Basics](https://rafalab.dfci.harvard.edu/dsbook/r-basics.html) <br> - [Intro to R](https://carpentry.library.ucsb.edu/CustomDC-R/02-intro-to-r/index.html) <br> - [Getting started with data](https://carpentry.library.ucsb.edu/CustomDC-R/03-starting-with-data/index.html)  | - Practice creating and storing objects <br> - Learn about different types of data that can be used <br> - How to import data <br> - How to install R packages |
| 3 | - Creating publication-ready figures with ggplot2 | - Practice graphing data in different ways <br> - Learn how to prepare figures that can be used for publications |
| 4 | - Comparing multiple variables between groups | - Practice testing whether species/groups are different and what variables may be causing those differences <br> - Understand the difference between multivariate and univariate statistical tests, and how they complement one another |
| 5 | - Using GLMMs to handle more complex datasets | - Understand the benefits of using GLMMs or GAMs rather than ANOVAs and t-tests <br> - Understand what pseudoreplication is and how to avoid it <br> - Learn approaches to determine which model or set of models best explain the variation in your data |
